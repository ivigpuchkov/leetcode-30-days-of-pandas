# leetcode-30-days-of-pandas

В этом репозитории собраны мои решения всех задач из тренажера LeetcCode (https://leetcode.com/studyplan/30-days-of-pandas/). 
Весь трек был успешно пройден за 1.5 недели.

## Стек
Все задачи были решены бибилиотекой **Pandas**
В процессе решения были применены следующие инструменты и методы:
* Фильтрация по сложным условиям, сортировка датафреймов.
* Работа с текстовыми данными.
* Регулярные выражения в Pandas.
* Объединение таблиц.
* Группировка и агрегирование, оконные функции.
* Свобдные таблицы.
